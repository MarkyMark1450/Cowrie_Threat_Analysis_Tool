# Cowrie Threat Analysis Tool

## Description

This project is a Python-based threat analysis tool designed to analyze log data generated by the Cowrie SSH honeypot.
The tool parses Cowrie JSON logs, performs behavioral analysis on attacker activity, generates visualizations, and produces a summary report.

The goal of this project is to demonstrate practical cybersecurity log analysis using Python, pandas, and matplotlib.

## Features

* Parse Cowrie honeypot JSON logs
* Analyze attacker IP activity
* Detect brute-force login attempts
* Identify suspicious command usage
* Session duration analysis
* Activity by hour visualization
* Graph generation with matplotlib
* Text report generation
* Command-line interface with argparse

## Installation

Create virtual environment (optional but recommended):

```
python -m venv .venv
```

Activate:

Windows / Git Bash:

```
source .venv/Scripts/activate
```

Install dependencies:

```
pip install -r requirements.txt
```

## Usage

Run analysis only:

```
python src/main.py
```

Generate graphs:

```
python src/main.py --graphs
```

Generate report:

```
python src/main.py --report
```

Generate both:

```
python src/main.py --graphs --report
```

## Output

Graphs are saved to:

```
output/graphs/
```

Report is saved to:

```
output/reports/cowrie_threat_report.txt
```
